297: Serialize and Deserialize Binary Tree
Description: Design an algorithm to serialize and deserialize a binary tree. 
Serialization is converting a tree into a single string representation. 
Deserialization is converting that string back into the original tree structure.

Intuition: Think of this as taking a layered snapshot of the tree. A standard Breadth-First Search (BFS / Level-Order Traversal) visits nodes layer-by-layer. 
If we capture every single node and explicitly record NULL states (using a marker like #), the resulting flat string uniquely represents that exact tree shape. 
When reconstructing, we use a queue to track parent nodes in the same exact order, popping a parent and immediately assigning its next two string tokens as its left and right children.

Approach:
serialize (Tree to String):Initialize a classic BFS queue and push the root.While the queue isn't empty, pop a node.If the node is NULL, append #, to your string.If the node is valid, append its value followed by a comma (e.g., 1,), and unconditionally push both its left and right children into the queue (even if they are NULL).
deserialize (String to Tree):Feed the string into a C++ stringstream and parse the first token using getline(s, str, ',') to create the main root.Push the root into a reconstruction queue.While the queue is not empty, pop the front node.Read the next token for the left child: If it's #, set node->left = NULL. Otherwise, create a new node, attach it to node->left, and push it to the queue.Read the following token for the right child: If it's #, set node->right = NULL. Otherwise, create a new node, attach it to node->right, and push it to the queue.

Complexity:

Time Complexity: $O(N)$ — Both functions process every node and structural empty child slot (NULL) exactly once.
Space Complexity: $O(N)$ — The queue at its peak stores the widest level of the tree (up to $N/2$ nodes), and the string/stringstream allocates memory proportional to the size of the tree.

Code:
{
public:
    // Encodes a tree to a single string.
    string serialize(TreeNode* root) {
        if (!root) return "";
        
        queue<TreeNode*> q;
        q.push(root);
        string ans;
        
        while (!q.empty()) {
            TreeNode* node = q.front();
            q.pop();
            
            if (node == NULL) {
                ans.append("#,");
            } else {
                ans.append(to_string(node->val) + ',');
                q.push(node->left);
                q.push(node->right);
            }
        }
        return ans;
    }

    // Decodes your encoded data to tree.
    TreeNode* deserialize(string data) {
        if (data.length() == 0) return NULL;
        
        stringstream s(data);
        queue<TreeNode*> q;
        string str;
        
        // Setup the root node
        getline(s, str, ',');
        TreeNode* root = new TreeNode(stoi(str));
        q.push(root);
        
        while (!q.empty()) {
            TreeNode* node = q.front();
            q.pop();

            // 1. Reconstruct Left Child
            getline(s, str, ',');
            if (str == "#") {
                node->left = NULL;
            } else {
                TreeNode* leftNode = new TreeNode(stoi(str));
                node->left = leftNode;
                q.push(leftNode);
            }

            // 2. Reconstruct Right Child
            getline(s, str, ',');
            if (str == "#") {
                node->right = NULL;
            } else {
                TreeNode* rightNode = new TreeNode(stoi(str));
                node->right = rightNode; // Fixed the copy-paste pointer bug!
                q.push(rightNode);
            }
        }
        return root;
    }
};
