## 🛠️ Setup & Quick Start

### Prerequisites
```
✓ C++ compiler (g++ or cl.exe)
✓ MinGW (Windows) or GCC (Linux/Mac)
✓ VS Code with C++ extension
✓ Basic C++ knowledge
```

### Compile & Run
```bash
# Navigate to any DAY-XX folder
cd DAY-36

# Compile
g++ -o quickSort quickSort.cpp

# Run
./quickSort
```

### VS Code Setup
1. Install C/C++ extension: `ms-vscode.cpptools`
2. Press `Ctrl+Shift+B` to build
3. Run the executable from terminal
