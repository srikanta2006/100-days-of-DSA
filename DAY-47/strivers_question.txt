Children Sum Property in a Binary Tree
Intuition:
We can only increase node values. 
To guarantee that a parent's value equals the sum of its children without falling short, we must first pass any large parent values down to the children. 
Once we reach the bottom of the tree, we recalculate the sums and pass the correct totals back up to the parents.

Approach:
Base Case: If the node is NULL or a leaf node, return (leaf nodes don't need updates).
Calculate Initial Child Sum: Safely add the values of the left and right children (if they exist).
Push Down (Pre-order):If child_sum < root->data: We are short on value! Overwrite both existing children with the root's value to ensure we don't lose the total.
If child_sum >= root->data: Overwrite the root with the child_sum.
Recurse: Call the function on root->left and root->right.
Pull Up (Post-order): Now that the children are fully updated and valid, recalculate their sum and update root->data = new_left + new_right.

Complexity:
Time: 
O(N) — We visit every node exactly once.
Space: 
O(N) — Worst-case recursive call stack (if the tree is a skewed straight line). 

Code:
    void changeTree(Node* root) {
        if(root == NULL) return;
        
        // 1. Gather child sum
        int childSum = 0;
        if(root->left) childSum += root->left->data;
        if(root->right) childSum += root->right->data;
        
        // 2. Push Down
        if(root->left != NULL || root->right != NULL) { // Not a leaf
            if(childSum >= root->data) {
                root->data = childSum;
            } else {
                if(root->left) root->left->data = root->data;
                if(root->right) root->right->data = root->data;
            }
        }
        
        // 3. Recurse
        changeTree(root->left);
        changeTree(root->right);
        
        // 4. Pull Up (Backtrack)
        int finalSum = 0;
        if(root->left) finalSum += root->left->data;
        if(root->right) finalSum += root->right->data;
        
        if(root->left != NULL || root->right != NULL) { // Not a leaf
            root->data = finalSum;
        }
    }
};
